Remove Duplicate Occurrences

you are given a string (STR) of length N, consisting of only the lowercase english alphabet. Your task is to remove all the dplicate occurrences of characters in the string.

Solution:

        let input = "abcadeecfb";
        let output=new Set();
        let result="";
        for (let i=0;i< input.length;i++){
            output.add(input[i]);
        }
        for (let ltr of output.keys()){
            result+=ltr;
        }
        console.log(result);
Result :
        Input
        "abcadeecfb"
        Output
        "abcdef"

Count of Alphabet (map)

You are given a string (STR) of length N, you have to print the count of all alphabet.(using maps)

Solution:

let myMap = new Map();
        let s="abcadeecfb";
        for(let i=0;i< s.length;i++){
            if(myMap.has(s[i])){
            let previousCount=myMap.get(s[i]);
            myMap.set(s[i],previousCount+1);
            }
            else{
            myMap.set(s[i],1);
            }
        }
        for(let [ch,freq] of myMap.entries()){
            console.log(ch+"="+freq);
        }
Result :
        Input
        "abcadeecfb"
        Output
        "a=2"
        "b=2"
        "c=2"
        "d=1"
        "e=2"
        "f=1"
