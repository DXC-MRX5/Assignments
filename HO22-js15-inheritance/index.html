Q1. Create 2 objects - parent and child. Attach methods to parent and use those methods in child object using parents prototype.

Solution:

        let parent = {
            name:"object1",
            greet: function(){
                console.log("Hi! "+this.name+" Welcome to Prepbytes.");
            }
        }
        let child=Object.create(parent);
        child.name="object2";
        child.greet();
Result :
        Hi! object2 Welcome to Prepbytes.

Q2. Write code to explain prototype chaining.

Solution:

        let grandfather={
            name: "someone-1",
            intro: function(){
                console.log("Hi! I am "+this.name);
            }
        }
        let father={
            name: "someone-2",
            __proto__:grandfather
        }
        let son={
            name: "someone-3",
            __proto__:father
        }
        son.intro();
Result :
        Hi! I am someone-3
     Prototype chaining is a mechanism in JavaScript that allows objects to inherit properties and methods
    from other objects. In this example, three objects are define 'grandfather', 'father' and 'son'. Then 'grandfather' is set 
    as the prototype of 'father' and again 'father' is set as the prototype of 'son'. Now 'son' inherits 'intro' property from 
    'father' and 'father' get that from 'grandfather'.

Q3. Add a method to calculate sum of all elements in Array in array's protype, use that method to calculate sum for multiple arrays.

Solution:

    Array.prototype.sum = function() {
        let sum = 0;
        for (let i = 0; i < this.length; i++) {
            sum += this[i];
        }
        return sum;
    }

    const array1 = [1, 2, 3, 4];
    const array2 = [5, 6, 7, 8];
    const array3 = [9, 10, 11, 12];

    const sum1 = array1.sum();
    const sum2 = array2.sum();
    const sum3 = array3.sum();

    console.log(sum1);
    console.log(sum2);
    console.log(sum3);
Result :
        10
        26
        42
