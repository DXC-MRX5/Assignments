Q7. Write a function that accepts parameter n and returns factorial of n.

Solution:

        let factorial = 1;
        function getFactorial(n){
            for(let i=1; i<=n; i++){
                factorial*=i;
            }
            console.log(factorial);
        }
        getFactorial(7);
Result :
        5040
