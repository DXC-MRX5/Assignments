1. Find the number of digits

Take a number from the user and print the count of digits in that number.

Solution:

    const Find_Digits = (N) => 
    {let count = 0;
    while(N !==0){//until N get 0
        N/=10;//get last digit after dec-point
        N=Math.floor(N);//reduce length of N
        count++;
    }
        return count;
    };
Result :
    Input
    1234
    Output
    4

2. Find the Fives

Among all the digits from 0-9, PrepBuddy likes number 5. He has a number and wants you to find out the number of times 5 occurred in the given number.

Solution:

    const Find_Five = (N) => 
    {let count =0;
        while (N !==0){
            if (N%10===5){
            count++;
            }
        N/=10;
        N=Math.floor(N);
        }	
        return count;
    };
Result :
    Input
    345654
    Output
    2

3. Find Sum

You are given an integer N, and your task is to find the sum of all the even integers starting from 1 upto N (N inclusive).

Solution:

    const findSum = (n) => {
    let ans =0;
        for(let it=1;it<=n;it++){
            if(it%2===0){//check even
                ans+=it;//add even
            }
        }
        return ans;  
    };
Result :
    Input
    6
    Output
    12

5. Print the Odds

Write a program which takes a number from user and print all odd numbers in between 2 and N, but print 2 first.

Solution:

    const Print_Odd = (N) => 
    {let p=2;
        while (p<=N) {
            if (p==2){//print 2 first
                console.log(2);
            }
            else if (p%2 ==1){//check odd
                console.log(p);
            }
        p++;
        }
    };
Result :
    Input
    10
    Output
    2
    3
    5
    7
    9

7. Check whether a Number is a prime or not.

Write a program which takes a number from user and check whether number is prime number or not a prime number.

Solution:

    const Prime_Check = (N) => 
    {let count=0;
	    for (let dvsr=1; dvsr<=N; dvsr++){
	        if (N%dvsr===0){
	            count++;
	        }
	    }
	
	    if (count==2){
	        return "YES";
	    }
	    else{
	        return "NO";
	    }
    };
